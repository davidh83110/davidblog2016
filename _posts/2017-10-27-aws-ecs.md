<span lang="EN-US"># aws ec2 run-instances --image-id ami-0797ea64 --count 1 \<o:p></o:p></span></div>
<div class="MsoNormal" style="margin-left: 24.0pt; mso-para-margin-left: 2.0gd; text-indent: 24.0pt;">
<span lang="EN-US">--instance-type t2.micro --key-name david
\<o:p></o:p></span></div>
<div class="MsoNormal" style="margin-left: 24.0pt; mso-para-margin-left: 2.0gd; text-indent: 24.0pt;">
<span lang="EN-US">--user-data file://user-data.txt <o:p></o:p></span></div>
<div class="MsoNormal" style="margin-left: 24.0pt; mso-para-margin-left: 2.0gd; text-indent: 24.0pt;">
<span lang="EN-US">--subnet-id $SUBNET --security-group-ids
$SECURITY_GROUP \<o:p></o:p></span></div>
<div class="MsoListParagraph" style="margin-left: 48.0pt; mso-para-margin-left: 4.0gd;">
<span lang="EN-US">--iam-instance-profile
Arn=arn:aws:iam::990090895087:instance-profile/ecsInstanceRole \<o:p></o:p></span></div>
<div class="MsoNormal" style="margin-left: 48.0pt; mso-para-margin-left: 4.0gd;">
<span lang="EN-US">--tag-specifications 'ResourceType=instance,Tags=[{Key=Name,Value=ECS-Instance-CLUSTER_NGINX}]'<o:p></o:p></span></div>
<div class="MsoNormal" style="margin-left: 48.0pt; mso-para-margin-left: 4.0gd;">
<span lang="EN-US"><br /></span></div>
<div class="MsoNormal" style="margin-left: 48.0pt; mso-para-margin-left: 4.0gd;">
<span lang="EN-US">```</span></div>
<div class="MsoNormal" style="margin-left: 48.0pt; mso-para-margin-left: 4.0gd;">
<br /></div>
<div class="MsoNormal">
<b style="mso-bidi-font-weight: normal;"><span lang="EN-US">Note:<o:p></o:p></span></b></div>
<div class="MsoNormal">
<span lang="EN-US">--Image-id ami-0797ea64 : Amazon Linux AMI<o:p></o:p></span></div>
<div class="MsoNormal">
<span lang="EN-US">--Iam-instance-profile Arn : use
ecsInstanceRole to connect cluster and instance, create this role if doesn’t

<div class="MsoNormal">

<div class="MsoNormal">

<div class="MsoNormal">
<span lang="EN-US"><o:p><br /></o:p></span></div>
<div class="MsoNormal">
<b style="mso-bidi-font-weight: normal;"><span lang="EN-US" style="font-size: 14.0pt;">Step10. Create an Classic Load Balancer (ELB) for
Nginx Cluster<o:p></o:p></span></b></div>
<div class="MsoNormal">
<br /></div>
<div class="MsoNormal">
<span lang="EN-US"><o:p>```</o:p></span></div>
<div class="MsoNormal">
<span lang="EN-US"># aws elb create-load-balancer
--load-balancer-name $ELB_NAME \<o:p></o:p></span></div>
<div class="MsoNormal">
<span lang="EN-US"><span style="mso-spacerun: yes;">&nbsp;</span>--listeners
"Protocol=HTTP,LoadBalancerPort=80,InstanceProtocol=HTTP,InstancePort=80"
\<o:p></o:p></span></div>
<div class="MsoNormal">
<span lang="EN-US"><span style="mso-spacerun: yes;">&nbsp;</span>--subnets $SUBNET_A $SUBNET_B
--security-groups $SECURITY_GROUP<o:p></o:p></span></div>
<div class="MsoNormal">
<span lang="EN-US">```</span></div>
<div class="MsoNormal">
<br /></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<b style="mso-bidi-font-weight: normal;"><span lang="EN-US">Note:<o:p></o:p></span></b></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US">In ELB, We don’t
have to specify target group.<o:p></o:p></span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<br /></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US"><o:p><br /></o:p></span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US"><o:p><br /></o:p></span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<b style="mso-bidi-font-weight: normal;"><span lang="EN-US" style="font-size: 14.0pt;">Step11. Register instance to
ELB<o:p></o:p></span></b></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<br /></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US"><o:p>```</o:p></span></div>
<div class="MsoNormal">
<span lang="EN-US"># aws elb
register-instances-with-load-balancer --load-balancer-name $ELB_NAME \<o:p></o:p></span></div>
<div class="MsoNormal">
<span lang="EN-US"><span style="mso-spacerun: yes;">&nbsp;</span>--instances $INSTANCE_ID_NGINX<o:p></o:p></span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US"><o:p>&nbsp;```</o:p></span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<br /></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<br /></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<b style="mso-bidi-font-weight: normal;"><span lang="EN-US" style="font-size: 14.0pt;">Step12. Build a new version for
Nginx image<o:p></o:p></span></b></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US">Because we should
specify the proxy_pass setting to Back-End service’s ALB DNS name in Nginx
config.<o:p></o:p></span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<br /></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US"><span style="background-color: #f4cccc;">Doclerfile:</span><o:p></o:p></span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<br /></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US"><o:p>```</o:p></span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US">FROM
990090895087.dkr.ecr.ap-southeast-1.amazonaws.com/david-nginx:latest<o:p></o:p></span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US">RUN sed -i
'52c\<span style="mso-spacerun: yes;">&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; </span>proxy_pass
http://$ALB_DNS_NAME:8080;' /etc/nginx/nginx.conf<o:p></o:p></span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US">EXPOSE 80<o:p></o:p></span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US">CMD (tail -F
/var/log/nginx/access.log &amp;) &amp;&amp; exec nginx -g \"daemon
off;\"<o:p></o:p></span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US"><br /></span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US">```</span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<br /></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US">Save Dockerfile
and build a new version then push it to ECR<o:p></o:p></span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<br /></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US">```# docker build -t
david-nginx:latest .```<o:p></o:p></span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US">```# docker tag
david-nginx:latest 990090895087.dkr.ecr.ap-southeast-1.amazonaws.com/david-nginx:latest```<o:p></o:p></span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US">```# docker push
990090895087.dkr.ecr.ap-southeast-1.amazonaws.com/david-nginx:latest```<o:p></o:p></span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<br /></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<b style="mso-bidi-font-weight: normal;"><span lang="EN-US">Note:<o:p></o:p></span></b></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US">We should use
`exec nginx –g "daemon off; "` to start Nginx, or Nginx process will
crash when ECS start.<o:p></o:p></span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US">Execute `aws ecs
get-login` to login if push image to ECR fail.<o:p></o:p></span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<br /></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<br /></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US"><o:p><br /></o:p></span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<b style="mso-bidi-font-weight: normal;"><span lang="EN-US" style="font-size: 14.0pt;">Step13. Create a task
definition for Nginx Service<o:p></o:p></span></b></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<br /></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US">```</span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US"># aws ecs
register-task-definition --network-mode host --family $TASK_NAME \<o:p></o:p></span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US"><span style="mso-spacerun: yes;">&nbsp;</span>--volumes
name=test-nginx,host={sourcePath=/tmp/container_tmp} --container-definitions \<o:p></o:p></span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US">"[{"name":"task-nginx","image":"$NGINX_DOCKER_IMAGE","cpu":50,"memory":500,"essential":true,
"portMappings":[{"hostPort":80,"containerPort":80,"protocol":"tcp"}],
\<o:p></o:p></span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US">"logConfiguration":{"logDriver":"awslogs","options":{"awslogs-group":"/ecs/david-nginx",
\<o:p></o:p></span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US">"awslogs-region":"ap-southeast-1","awslogs-stream-prefix":"nginx"}},
\<o:p></o:p></span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US">"mountPoints":[{"sourceVolume":"test-nginx","containerPath":"/tmp","readOnly":false}]}]"<o:p></o:p></span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US"><br /></span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US">```</span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<br /></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<b style="mso-bidi-font-weight: normal;"><span lang="EN-US">Note:<o:p></o:p></span></b></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US">--ntework-mode
host : Use host mode to make sure one instance only have one nginx container,<o:p></o:p></span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US">because ELB doesn’t
allow we use random port in ECS to point to ELB Listener just like ALB.<o:p></o:p></span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US">--volumes name :
Volume name , host={sourcePath=$YOUR_INSTANCE_PATH }<o:p></o:p></span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US">--container-definitions
{["name":"$TASK_NAME"]} must the same with --family name<o:p></o:p></span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US">logConfiguration:
awslogs – It will send stdout result to CloudWatch<o:p></o:p></span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US">mountPoints:
sourceVolume must the same with Volume name<o:p></o:p></span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US"><br /></span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US"><br /></span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US"><br /></span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<b style="mso-bidi-font-weight: normal;"><span lang="EN-US" style="font-size: 14.0pt;">Step14. Create a Nginx
Service for Nginx Cluster just like Step7<o:p></o:p></span></b></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<br /></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US">```</span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US"># aws ecs
create-service --service-name service-nginx --cluster $CLUSTER_NGINX \<o:p></o:p></span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US"><span style="mso-spacerun: yes;">&nbsp;</span>--task-definition $NGINX_TASK_DEFINITION
--desired-count 1 --load-balancers \<o:p></o:p></span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US"><span style="mso-spacerun: yes;">&nbsp;</span>"[{"loadBalancerName":"$ELB_NAME","containerName":"$CONTAINER_NEM","containerPort":80}]"
\<o:p></o:p></span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US"><span style="mso-spacerun: yes;">&nbsp;</span>--role ecsServiceRole<o:p></o:p></span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US"><br /></span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US">```</span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<br /></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<b style="mso-bidi-font-weight: normal;"><span lang="EN-US">Note:<o:p></o:p></span></b></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US">--load-balancers :
Use ELB that Step10 create<o:p></o:p></span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US">containerName must
the same with family name that Step13 you create<o:p></o:p></span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<span lang="EN-US">--role
ecsServiceRole : Use ecsServiceRole to let cluster to access EC2, S3,
CloudWatch (Can be modifity in IAM Settings)<o:p></o:p></span></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
<br /></div>
<div class="MsoNormal" style="tab-stops: 40.1pt;">
